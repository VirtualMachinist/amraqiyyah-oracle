# Panel
The Oracle's fundamental container — a bordered card on panel fill with an optional serif title. Accent recolours the border + title to signal a state.

```jsx
<Panel title="The six coordinates">…</Panel>
<Panel title="State 3 — The Unmoved Axis" accent="gold">…</Panel>
```

Accents: `gold` (Unmoved Axis), `teal` (Stellar Court), `copper`, `crimson`.
