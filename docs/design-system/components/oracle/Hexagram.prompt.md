# Hexagram
The six-line figure, drawn top (line 6) to bottom (line 1). Yang = solid gold bar, yin = split indigo; a moving line burns crimson with a dot.

```jsx
<Hexagram number={55} name="Abundance" lines={[true,true,false,true,false,false]} moving={[3,4]} upper="Thunder" lower="Fire" suit="Fire" divineName="Al-Muhyi" title="The situation"/>
```

`lines`: 6 booleans, index 0 = bottom line. `moving`: line numbers 1–6.
