# SegmentedControl
Framed multiple-choice selector for a small fixed set — reading Mode (A/B/C), book (Qur'ān/Zabūr). Selected segment lifts to a copper-bordered sunken fill.

```jsx
<SegmentedControl options={[{value:"A",label:"A — Astronomical"},{value:"B",label:"B — Deck"}]} value={mode} onChange={setMode}/>
```
